# Asset provenance

- `public/brand/korda-mark.png` is the approved Korda product mark, developed for this project from a user-directed visual concept.
- `build/icon.png` is the Linux package icon derived from the same approved mark.
- The Korda name and logo are product identity assets. Apache-2.0 licenses the source code, not trademark use.
- `public/browser-fixture.html` is a local test fixture created for Korda.
- Public landing screenshots depict Korda's own interface with fictional or empty data. They are maintained in the separate `Teolfeu/korda-site` repository.

Internal design explorations, QA captures and promotional-video working files are intentionally excluded from the public source repository.
